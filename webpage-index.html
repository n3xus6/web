Some special characters: <, >, &, ©, €

  1. HTML's job is to give text structure and meaning so that browsers can display it.
  2. To control the presentation of the page use CSS.
  • Use <em> or <strong> to give text a meaning. Screen readers can understand this. But don't use <b> for a bold look. Instead define the look in a CSS stylesheet.
